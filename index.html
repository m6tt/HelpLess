HelpLess by m6tt

Library of Less Helper functions

Dependencies

less.js, lesscss.org

Install

Follow the instructions on lesscss.org to get less up and running. @import helpless.less as you would any other .css or .less file.

You can download this project in either zip or tar formats.

$ git clone git://github.com/m6tt/HelpLess
