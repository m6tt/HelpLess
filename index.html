HelpLess

HelpLess is a Helper Library for the increasingly popular LESS dynamic stylesheet language. Read more about LESS here.

Simply import helpless.less and let it make your life easier.

How to import

: Description

Importing HelpLess into your LESS file is easy.

Code

                
  @import "libs/helpless.less";
